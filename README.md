# 郵局考古題練習網站（專業職（二）內勤）

純前端、可離線的考古題練習網站，自己讀書用。

## 怎麼開啟

最簡單：直接用瀏覽器（Chrome / Edge）打開 `index.html`（雙擊即可）。
所有作答紀錄、錯題、收藏都存在你這台電腦的瀏覽器裡。

## 功能

- **分科練習**：依科目逐題練，作答後立刻看正解與詳解
- **隨機練習**：跨科隨機抽題
- **模擬考**：可設定題數與作答時間，最後才結算分數
- **錯題複習**：做錯的題目自動進錯題本，可重練
- **我的收藏**：作答時點星星收藏重點題
- **學習統計**：總正確率、各科正確率

## 怎麼新增 / 替換考古題（重點）

編輯 `data/questions.js`，在 `questions: [ ... ]` 陣列裡照格式加題目即可。
檔案最上方有完整格式說明。一題範例：

```js
{
  id: "ch-101",                 // 唯一代號，不可重複
  subject: "chinese",           // 科目代號：chinese / english / management / postal
  year: 113,                    // 年度
  stem: "下列何者……？",
  options: ["選項A", "選項B", "選項C", "選項D"],
  answer: 2,                    // 正確選項索引：A=0, B=1, C=2, D=3
  explanation: "詳解文字（可留空）"
}
```

存檔後重新整理網頁就會生效。

> 目前題庫內全部是「範例題」（year 標為「範例」），請陸續換成真正的歷年考古題。
> 自己讀書用沒有版權問題；若日後要公開分享，請留意他人編寫的詳解版權。

## 之後想放到手機 / 公開（選用）

把整個資料夾上傳到免費靜態託管即可：
- **GitHub Pages**、**Netlify**、**Cloudflare Pages** 都可免費
- 用 http(s) 開啟時會啟用 PWA（`manifest.json` + `sw.js`），可「加到主畫面」並離線使用
- 記得補上 `icons/icon-192.png` 與 `icons/icon-512.png` 兩個圖示（目前未附）

## 檔案結構

```
post-exam/
├─ index.html          首頁
├─ manifest.json       PWA 設定
├─ sw.js               離線快取（託管時生效）
├─ css/style.css       樣式
├─ js/app.js           程式邏輯
└─ data/questions.js   ★ 題庫（你主要會編輯這個）
```
